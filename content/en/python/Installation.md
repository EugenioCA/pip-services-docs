### Intallation procedure

PIP.Services needs to be installed per module. The general form is:

```bash
pip install module-name
```
### Examples:

#### Commons module

```bash
pip install pip-services3-commons
```
 
#### Components module 
```bash
pip install pip-services3-components
```
