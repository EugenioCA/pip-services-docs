### Description
 
The TypeCode class contains the codes for data types that are used by the class [TypeConverter](../type_converter). 

### Enumeration members

The codes are as follows:

- **Unknown** = iota
- **String** = iota
- **Boolean** = iota
- **Integer** = iota
- **Long** = iota
- **Float** = iota
- **Double** = iota
- **DateTime** = iota
- **Duration** = iota
- **Object** = iota
- **Enum** = iota
- **Array** = iota
- **Map** = iota
